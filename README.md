# The-Date-Abase

### Welcome to the Date-Abase!

- **What is the Data-Abase?** The Data-Abase is a simple, interactive one-page website that helps users to find different date ideas from A to Z. Each idea is categorized by solo, friends, and/or romantic outings.

### Live Demo
You can view the live, hosted website directly in your browser:
[View Live Site](kholla05.github.io/The-Date-Abase/)

### Tech Stack
- **HTML5:** Semantic structure for the layout and content.
- **CSS3:** Responsive styling and categorization theme colors.

### Key Features
- **A-Z Index/Directory:** Located at the top of the page, this feature allows users to browse unique activities mapped to every letter of the alphabet.
- **Category Tags:** Clear tags that show which dates are best suited for each group (Solo, Friend, or Romantic)
